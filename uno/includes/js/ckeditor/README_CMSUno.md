CKEditor
--------

CMSUno use a custom version of CKEditor made on https://ckeditor.com/ with the Standard Package plus :

* BR CLEAR
* COLOR BUTTON
* DIV CONTAINER MANAGER
* FIND / REPLACE
* FONT SIZE AND FAMILY
* JUSTIFY
* SHOW BLOCKS

SKIN : MOONO-LISA

LANG : EN, FR, SP

Nothing more.
